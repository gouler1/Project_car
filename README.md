# Система аренды автомобилей

Веб-приложение для аренды авто с учётом доступности и платежами.

## Установка
1. Клонировать repo: `git clone <url>`
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить: `flask run`

## Функциональности
- Регистрация/логин
- Просмотр доступных авто
- Бронирование с проверкой дат
- Симулированные платежи

## Стек
- Flask, SQLAlchemy, Flask-Login, WTForms, Bootstrap
